# Shiwani Mishra - Portfolio

Personal portfolio and blog built with Next.js.

Live site: https://shiwani42.github.io/

## About

This site presents the profile of **Shiwani Mishra**:

- Co-Founder at Heisenbug
- Electrical Engineering (Dual Degree) at IIT Bombay
- Minor in Computer Science and AI
- Focused on agentic compliance and security-first systems

All profile content is sourced from [`src/data/resume.tsx`](./src/data/resume.tsx).

## Highlights

- Blog posts
- Projects
- Sections: Work Experience, Education, Skills, Projects, Volunteering, Contact

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- content-collections (MDX blog)

## Local Development

1. Install dependencies:

```bash
pnpm install
```

2. Run the dev server:

```bash
pnpm dev
```

3. Open http://localhost:3000

## Build

```bash
pnpm build
```

This project is configured for static export and GitHub Pages deployment.
